Firebase Setup

TaskBoardにはFirebase Realtime Databaseが必要です。

1Firebase プロジェクト作成

Firebase Console でプロジェクト「task-board」を作成し、Realtime Database を有効化してください（asia-southeast1 推奨）。

2Firebase Config を貼り付け

Web アプリを追加し、firebaseConfig オブジェクトを JSON で貼り付けてください:
